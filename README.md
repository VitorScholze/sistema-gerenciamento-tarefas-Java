# sistema-gerenciamento-tarefas
# 📝 Gerenciador de Tarefas em Java

## 📌 Sobre

Aplicação de linha de comando (CLI) para gerenciamento de tarefas, desenvolvida em Java.

Permite criar, listar, concluir e remover tarefas utilizando conceitos de programação orientada a objetos.

---

## 🚀 Funcionalidades

* Adicionar tarefas
* Listar tarefas
* Concluir tarefas por ID
* Remover tarefas por ID

---

## 🧠 Conceitos aplicados

* POO (Encapsulamento)
* ArrayList
* Stream API
* Estruturas de repetição e decisão

---

## 🎯 Objetivo

Projeto desenvolvido para praticar Java e consolidar conceitos fundamentais de programação.
